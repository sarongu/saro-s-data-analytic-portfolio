# saro-s-data-analytic-portfolio
Data Analytics Learning Portfolio

This repository documents my ongoing learning and project work in data analytics using publicly available datasets.

The goal of this portfolio is to strengthen my skills in:
- SQL
- Power BI
- Exploratory Data Analysis (EDA)
- Applied Statistics
- Data Visualization
- Public sector data analysis

All datasets used in this repository come from publicly available sources. No confidential or proprietary data is included.

---

## Portfolio Goals

This project-based portfolio is designed to help me:

- Build fluency in applied statistical analysis
- Practice real-world data workflows
- Develop dashboards and insights using Power BI
- Improve SQL querying and data modeling
- Document analytical thinking and decision-making

Each project focuses on a specific analytical question and highlights different statistical techniques.
